{\footnotesize\ttfamily void u\+\_\+perror (\begin{DoxyParamCaption}\item[{char \texorpdfstring{$\ast$}{*}}]{message }\end{DoxyParamCaption})}



This is an analogue of the perror(3) function. This allows the shell or other function to pass in a message describing what error occured, which the function then concatenates to the default error message based on your errno value. The filesystem and kernel will both set errno and return -\/1 on system calls if they fail, after which \doxylink{error_8h_ab2e01b83bcc336abdcc77b088661cdfd}{u\+\_\+perror()} can be called.

{\footnotesize\ttfamily int errno\hspace{0.3cm}{\ttfamily [extern]}}
