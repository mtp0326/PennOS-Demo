 Welcome to Penn\+OS, a project designed to simulate the functionalities of a UNIX-\/like operating system. This project was created by Aaron Tsui, Matt Park, Joesph Cho, and Maya Huizar.\hypertarget{index_autotoc_md0}{}\doxysubsection{\texorpdfstring{Overview}{Overview}}\label{index_autotoc_md0}
Penn\+OS is a UNIX-\/like operating system that runs as a guest OS within a single process on a host OS. It includes implementations of a basic priority scheduler, a FAT file system (Penn\+FAT), and user shell interactions.\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Documentation}{Documentation}}\label{index_autotoc_md1}
