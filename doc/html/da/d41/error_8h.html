void u_perror ( char * message)

This is an analogue of the perror(3) function. This allows the shell or other function to pass in a message describing what error occured, which the function then concatenates to the default error message based on your errno value. The filesystem and kernel will both set errno and return -1 on system calls if they fail, after which u_perror() can be called.

Parameters
messageThis is the message that will be concatenated to the default error message.
